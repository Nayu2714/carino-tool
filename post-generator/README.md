# 画像差し替え時の注意

このフォルダの画像を差し替えるときは、以下を必ず守ってください。

## 1. 形式は必ず PNG（.png）
画像コピー機能は PNG 形式のみ対応しています。JPEG などを置くと「画像をコピー」が失敗します。
※拡張子だけ .png にしても、中身が JPEG だとダメです。ペイント等で「PNG として保存」し直してください。

## 2. ファイル名は変えない（同じ名前で上書きする）
ツールが下記の名前で画像を読み込んでいます。差し替えるときは同名で上書きしてください。

- pre/pre-poster.png … 事前応募ポスト用
- sameday/sameday-poster.png … 当日：ポスター画像
- sameday/sameday-rules.png … 当日：応募条件・注意事項画像

名前を変えたい場合は、post-generator/index.html 内の該当パス（img src と onclick）も一緒に直してください。
